# 🚗 CodeAlpha_Car_Price_Prediction_using_Machine_Learning

## 📌 Project Overview
This project predicts the selling price of cars using Machine Learning techniques.  
It analyzes important features like mileage, car age, fuel type, and present price to estimate the resale value.

This project follows a complete **data science workflow**:
- Data Preprocessing  
- Data Visualization  
- Model Building  
- Model Evaluation  

---

## 🎯 Objective
- Predict car prices accurately  
- Understand key factors affecting price  
- Apply regression models to real-world problems  

---

## ⚙️ Technologies Used
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🔄 Project Workflow

### 1️⃣ Data Preprocessing
- Removed unnecessary column (Car_Name)  
- Converted categorical variables using encoding  
- Created new feature: **Car Age**

---

### 2️⃣ Data Visualization 📊
Used advanced graphs:
- Histogram + KDE (Price Distribution)  
- Correlation Heatmap  
- Scatter Plot (Mileage vs Price)  
- Regression Plot (Car Age vs Price)  
- Pairplot  
- Boxplot (Outliers Detection)

---

### 3️⃣ Model Building 🤖
- Linear Regression  
- Random Forest Regressor  

---

### 4️⃣ Model Evaluation 📈
Metrics used:
- Mean Absolute Error (MAE)  
- Mean Squared Error (MSE)  
- R² Score  

Graphs:
- Actual vs Predicted Plot  
- Residual Plot  

---

## 🔍 Key Insights
- Older cars have lower prices  
- Higher mileage reduces value  
- Present price has strong impact on selling price  

---

## 🚀 Results
- Linear Regression gives baseline performance  
- Random Forest provides better accuracy  

---

## 🌍 Real-World Applications
- Used car resale platforms (OLX, Cars24, CarDekho)  
- Insurance valuation  
- Automobile analytics
